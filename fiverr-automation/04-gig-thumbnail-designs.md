# Gig Thumbnail Concepts - Canva Implementation

## Design Guidelines

### Universal Branding Elements
- **Background**: Bright gradient (teal → purple → pink)
- **Main Font**: Montserrat (Bold & Regular)
- **Accent Font**: Roboto (for technical details)
- **Color Scheme**: #1abc9c (teal), #8e44ad (purple), #e74c3c (red accent)
- **Style**: Modern, clean, professional with tech aesthetic

## Canva Template Specifications

### Universal Template Settings
- **Dimensions**: 1280 × 720 pixels (16:9 Fiverr ratio)
- **Color Mode**: RGB (not CMYK)
- **Resolution**: 100 dpi minimum
- **File Format**: PNG for upload, PSD for archive

## A/B Testing Strategy

**Test Variables**:
1. **Text vs Graphic heavy**
2. **Light vs Dark backgrounds**
3. **People vs Abstract concepts**
4. **Large vs Medium text**
5. **2 vs 3 color schemes**

**Testing Schedule**: Track each for 2 weeks, Fiverr's conversion data

**Success Metrics**: 
- Click-through rate (CTR) 
- Impressions to views ratio
- Inquiries generated
- Order conversion rate

**Optimization**: Update every 30-45 days based on performance data
