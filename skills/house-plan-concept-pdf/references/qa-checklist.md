# Concept Plan PDF QA Checklist

## Visual quality
- Title block present and readable
- Consistent font sizing (title, labels, dimensions)
- No overlapping labels
- Line weights differentiate outer walls vs partitions

## Plan completeness
- Exterior footprint dimensions shown
- Key room dimensions shown
- Carport/garage and porch included if requested
- Primary bath fixture style reflected in labels

## Elevation completeness
- Front elevation includes roofline, porch, entry, openings
- Rear elevation includes porch and openings
- Style cues match selected exterior style

## Accuracy guardrails
- Concept note and disclaimer present
- Any assumptions listed
- Do not imply engineering, code approval, or permit readiness
