## House Plan Generation Skill

Built `house-plan-concept-pdf` skill for Triumph Homes. Multiple iterations:
- v1-v6: Basic PDF sketches with room blocks
- v7-v8: Larger sheet format attempts
- v9-v12: Portrait letter format with title band, dimensions, grid bubbles

## Key Learnings

1. PDF generation libraries (ReportLab) are too primitive for professional architectural output
2. DXF is the right format - opens in LibreCAD, AutoCAD, any CAD software
3. Building deployable apps requires proper containerization and CI/CD setup
4. Multi-user auth requires database + session management + role middleware
